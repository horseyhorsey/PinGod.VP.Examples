Detailed Description

ADAM targets: 42K for the Teela switch Teela allows you to start a mode by the targets by hitting them only once. In Ripper Multiball the lights do not increase but can be used with Teela. Completing all ADAM targets and starting one mode is enough to light solid. You can bank the Teela switches by not going for the ADAM targets. Then if hit with all 3 Teela banked you start all 3.
